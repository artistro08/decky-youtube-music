# Spacing v2 Implementation Plan

> **For agentic workers:** REQUIRED: Use superpowers:subagent-driven-development (if subagents available) or superpowers:executing-plans to implement this plan. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Remove the container-level gap so all content stretches edge-to-edge, then add 12px left/right padding to individual items (excluding playback control buttons and queue item buttons which stay full-width).

**Architecture:** The previous plan added `padding: 0 12px` to the `Section` outer div — this is the visible gap in the screenshot. The fix is two-phase: (1) remove the Section outer padding so everything is edge-to-edge, and (2) wrap specific non-button items in `<div style={{ padding: '0 12px' }}>` individually. The `Section` title keeps its own inline left/right padding. The DOM traversal in `TabsContainer` (which zeros Decky's `TabContentColumn` padding) stays unchanged.

**Tech Stack:** React (TypeScript), `@decky/ui`, Rollup

**Items that stay full-width (no padding wrapper):**
- `Focusable` rows containing `DialogButton` controls: ⏮▶⏭ and 👍👎
- Queue item `Focusable` rows

**Items that get `padding: 0 12px` wrapper:**
- Album art container
- Track info container (title, artist text, seek `SliderField`)
- Volume `SliderField`
- Mute `ButtonItem`
- Shuffle `ToggleField`
- Repeat `ButtonItem`
- Queue loading/empty status divs
- Queue "Clear Queue" `ButtonItem`

## Chunk 1: Fix Section — remove outer padding, title keeps its own

### Task 1: Update `Section` to remove outer padding

**Files:**
- Modify: `src/components/Section.tsx`

Remove `padding: '0 12px'` from the outer div (back to no padding). The title div keeps left/right padding via its own inline style (`padding: '12px 12px 4px'`).

- [ ] **Step 1: Replace full file content**

```tsx
import type { ReactNode } from 'react';

interface SectionProps {
  title?: string;
  children: ReactNode;
}

export const Section = ({ title, children }: SectionProps) => (
  <div>
    {title && (
      <div style={{
        padding: '12px 12px 4px',
        fontSize: '11px',
        fontWeight: 'bold',
        textTransform: 'uppercase' as const,
        color: 'var(--gpSystemLighterGrey)',
        letterSpacing: '0.04em',
      }}>
        {title}
      </div>
    )}
    {children}
  </div>
);
```
