## Task 4: index.tsx — fix panel title header gradient

**Files:**
- Modify: `src/index.tsx`

The panel header has a gradient artifact caused by the `staticClasses.Title` element's box-shadow and top padding bleeding onto the back button. Fix by:
1. Removing `paddingTop` and `boxShadow` from the `titleView` div via inline style
2. Using a `useEffect` in `Content` to find the parent flex container and set `gap: '0'`

`useEffect` is already imported on line 3.

- [ ] **Step 1: Add inline styles to titleView**

Find:
```tsx
  titleView: <div className={staticClasses.Title}>YouTube Music</div>,
```

Replace with:
```tsx
  titleView: <div className={staticClasses.Title} style={{ paddingTop: '0', boxShadow: 'none' }}>YouTube Music</div>,
```

- [ ] **Step 2: Add useEffect to Content to zero the header flex gap**

Find:
```tsx
const Content = () => (
  <PlayerProvider>
    <PluginContent />
  </PlayerProvider>
);
```

Replace with:
```tsx
const Content = () => {
  useEffect(() => {
    const titleEl = document.querySelector(`.${staticClasses.Title}`);
    if (titleEl?.parentElement) {
      titleEl.parentElement.style.gap = '0';
    }
  }, []);
  return (
    <PlayerProvider>
      <PluginContent />
    </PlayerProvider>
  );
};
```

- [ ] **Step 3: Commit**

```bash
git add src/index.tsx
git commit -m "fix: remove title header box-shadow, top padding, and flex gap causing back button gradient"
```
