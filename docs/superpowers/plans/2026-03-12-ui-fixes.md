# UI Fixes Implementation Plan

> **For agentic workers:** REQUIRED: Use superpowers:subagent-driven-development (if subagents available) or superpowers:executing-plans to implement this plan. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Fix volume icon rendering, slider offset, full-width buttons, repeat button text+icon, and increase side padding from 12px to 16px.

**Architecture:** All changes are isolated to two component files (`PlayerView.tsx`, `QueueView.tsx`). No new files are created. All `PlayerView.tsx` edits must be complete before running `npm run build` because TypeScript's `noUnusedLocals: true` will fail the build if any intermediate state has unused locals.

**Tech Stack:** React TypeScript, `@decky/ui` (`SliderField`, `DialogButton`, `Focusable`, `ToggleField`), `react-icons`
